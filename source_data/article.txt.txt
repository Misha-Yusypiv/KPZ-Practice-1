GitHub - це веб-сервіс для зберігання та спільної роботи над проектами з використанням системи контролю версій Git.
Сервіс був створений у 2008 році Джоном Резігом та Томом Престон-Вернером.

GitHub - це інтернет-портал, в якому розробники та програмісти можуть завантажувати створений ними код та спільно працювати над його покращенням.
Відмінною особливістю GitHub є надійна система контролю версій.
Контроль версій дозволяє програмістам налаштовувати програмне забезпечення без шкоди для самого програмного забезпечення.

У червні 2018 року Microsoft оголосила про придбання GitHub за $7,5 млрд.
Це придбання показало зростаючий інтерес Microsoft до відкритого програмного забезпечення та розвитку відкритих стандартів.
